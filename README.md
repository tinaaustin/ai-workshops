## What Is UnBlooms?

UnBlooms is a practical framework for educators who want to preserve student thinking in an AI-rich learning environment. It builds on the insight that AI can produce polished work without necessarily producing learning. The goal is to help educators design for visible reasoning, metacognition, critical evaluation, and authentic human judgment.
